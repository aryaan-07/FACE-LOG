# FACE-LOG
Facial Recognition-Based Student Attendance System.


We have developed a Python-based face recognition attendance system that combines computer vision and
AI to automate a traditionally manual process. The system uses OpenCV for video capture and face
detection, the face_recognition library (deep CNN) for identification, and Tkinter for an operator GUI.
Tests show the system works in real time and accurately records attendance without manual intervention.
By replacing paper sheets or swipe cards, our approach improves security (preventing proxy attendance)
and saves time. The use of deep embeddings (128D feature vectors) ensures robust recognition.
In summary, this work illustrates how AI and image processing can be applied to a practical automation
problem in education and enterprise. The system aligns with trends in computer engineering (AI-driven
video analysis) and fulfills the conference themes of automation and image-based AI applications.
